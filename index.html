Заголовки в HTML

Заголовки визначаються тегами <h1> ... <h6>. Тег <h1> використовується для найважливішого заголовка, а <h6> – для найменш важливого.

Форматування тексту

Для виділення важливих частин тексту використовуються теги <b>, <strong>, <i>, <em>, а також інші елементи форматування.

Наприклад, цей текст виділений як важливий, а цей – підкреслює логічний наголос. Можна також використовувати підкреслення, підсвічування, надрядкові X2 та підрядкові H2 елементи.

«Пошукові системи використовують заголовки для індексації структури та змісту веб-сторінок. Користувачі ж швидко орієнтуються завдяки зрозумілим заголовкам».

Висновки

Коректне використання заголовків, форматування тексту, зображень та посилань робить веб-документи структурованими, зручними для читання і зрозумілими як для користувачів, так і для пошукових систем.

Пам'ятайте: заголовки слід використовувати лише для позначення структури, а не для візуального збільшення шрифту.
